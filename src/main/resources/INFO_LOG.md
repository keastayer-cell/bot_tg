# Telegram Bot - @tg_tanya_bot

## Описание

Бот для автоматической рассылки сообщений и картинок подписчикам в заданное время.

## Возможности

- **Авторассылка** — бот автоматически отправляет контент подписчикам в указанное время (по умолчанию 09:00)
- **Управление подписчиками** — админ может добавлять/удалять получателей рассылки
- **Очередь сообщений** — тексты и картинки добавляются в очередь и отправляются по очереди
- **Обратная связь** — подписчики могут отправлять сообщения админу
- **Статистика** — просмотр количества сообщений, картинок и получателей

---

## Команды для АДМИНА

### Основные команды

| Команда | Описание |
|---------|----------|
| `/start` | Показать список всех команд (только для админа) |
| `/recipients` | Показать список всех получателей |
| `/addrecipient ID` | Добавить получателя по chat ID |
| `/addrecipient @nick` | Добавить получателя по username |
| `/delrecipient ID` | Удалить получателя |
| `/time` | Показать время автоотправки |
| `/settime HH:MM` | Установить время отправки (например, `/settime 14:00`) |

### Работа с очередью

| Команда | Описание |
|---------|----------|
| `/add текст` | Добавить текстовое сообщение в очередь |
| `/queue` | Показать текущую очередь |
| `/fillqueue` | Заполнить очередь из сохранённых текстов и картинок |
| `/stats` | Показать статистику (тексты, картинки, очередь) |
| `/now` | Отправить сообщение/картинку прямо сейчас |

### Работа с картинками

| Команда | Описание |
|---------|----------|
| Отправить фото боту | Сохранить картинку и добавить в очередь |
| `/images` | Показать список сохранённых картинок |
| `/sendimage N` | Отправить картинку №N всем получателям |
| `/sendimageall` | Отправить все картинки всем получателям |

### Прочее

| Команда | Описание |
|---------|----------|
| `/messages` | Показать сохранённые текстовые сообщения |
| `/logs` | Показать последние логи бота |
| `/msg текст` | Отправить произвольное сообщение всем получателям |

## Команды для ПОДПИСЧИКОВ

| Команда | Описание |
|---------|----------|
| `/start` | Подписаться на рассылку |
| `📬 Админу` | Отправить сообщение админу |

Подписчик может только:
- Подписаться на рассылку
- Написать сообщение админу

---

## Как это работает

1. **Добавление контента:**
   - Админ отправляет текст с командой `/add Привет!` — текст добавляется в очередь
   - Админ отправляет фото боту — фото сохраняется и добавляется в очередь
   - Команда `/fillqueue` добавляет все сохранённые тексты и картинки в очередь

2. **Отправка:**
   - В указанное время (по умолчанию 09:00) бот берёт случайный элемент из очереди и отправляет всем подписчикам
   - Команда `/now` отправляет контент немедленно

3. **Подписка:**
   - Новый пользователь нажимает `/start` и автоматически подписывается на рассылку
   - Админ также может добавить вручную по ID или username

---

## Конфигурация

- **Время отправки:** настраивается через `/settime HH:MM`
- **Часовой пояс:** Europe/Moscow
- **Интервал проверки:** каждую минуту
